# 待办事项

## 创建待办事项

在顶部的输入框中输入内容，然后按 **Enter** 或点击**添加**。新待办事项将出现在列表底部。

## 子待办事项

从 `...` 菜单或右键上下文菜单中点击**添加子待办**。子待办事项在其父项下方缩进显示，可通过展开/折叠箭头图标控制显示。

删除父待办事项会同时删除其所有子待办。

## 完成待办事项

点击复选框切换完成状态。已完成的待办事项会以删除线样式显示。你始终可以切换自己的待办状态。切换他人的待办需要 `complete_any_todo` 权限。

## 内联编辑

从 `...` 菜单或上下文菜单中点击**编辑**。标题将变为输入框。按 **Enter** 保存，按 **Escape** 取消。

## 排序

在桌面端，通过左侧的拖拽手柄拖动待办事项进行重新排序。新顺序将通过排序 API 自动保存。移动端不支持拖拽排序。

## 评论

点击**评论**按钮（或评论数量标记）打开评论对话框。评论按待办事项分组显示，包含用户名、时间戳和内容。

## 多选操作

右键点击待办事项并选择**选择**，或使用 `...` 菜单。进入选择模式后：

- 点击条目切换选中状态
- **Shift+点击**进行范围选择
- 出现选择栏，提供批量操作：全选、标记完成、标记未完成、删除、清除选择

## 上下文菜单

右键点击任何待办事项可打开上下文菜单，包含：添加子待办、评论、选择/取消选择、全选、编辑、标记完成/未完成、删除。可用操作取决于你的权限。
