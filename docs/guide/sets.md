# Todo Sets

Todo sets are containers that group related todos together within a team. Every todo belongs to exactly one set.

## Default Set

When you first access a team, Glint automatically creates a default set named "Not Grouped" (configurable in Settings). This ensures you can always create todos immediately.

## Creating Sets

Click **New set** at the bottom of the sidebar set list. Type a name and press Enter or click the add button.

## Renaming Sets

Click the `...` menu on a set and choose **Rename**. A dialog will appear where you can enter the new name.

## Deleting Sets

Click the `...` menu on a set and choose **Delete**. This deletes the set and all todos within it (including sub-todos and comments).

## Reordering Sets

On desktop, drag sets in the sidebar to reorder them. The new order persists to the server.

## Per-Set Permissions

Each set can have its own permission overrides. In **Settings > Permissions**, select a specific set from the scope dropdown to configure which roles can view, create, edit, or delete todos in that set. Per-set overrides take priority over global permissions.
